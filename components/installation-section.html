Installation

Get up and running in minutes with multiple installation options

Install via pip (Recommended)

pip install torchmodular

The easiest way to install TorchModular. This will automatically handle all dependencies.

Install via conda

conda install -c conda-forge torchmodular

For conda users who prefer managing environments with conda.

Development Installation

git clone https://github.com/yourusername/torchmodular.git
cd torchmodular
pip install -e .

For developers who want to contribute or customize the toolkit.

Requirements

Python

Python 3.7 or higher

PyTorch

PyTorch 1.9.0 or higher

System

Windows, macOS, or Linux

Verify Installation

import torchmodular as tm
print(tm.__version__)

# Test basic functionality
model = tm.Linear(10, 5)
print("Installation successful!")
