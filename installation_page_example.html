Installation Guide

Get SoundByte up and running on your system

Prerequisites

  • Python 3.8 or higher
  • conda (for environment management)
  • CUDA-compatible GPU (optional, for GPU acceleration)

Setup Instructions

1. Extract and Navigate

unzip soundbyte_toolkit.zip cd soundbyte_toolkit_build

2. Create Conda Environment

conda create -n soundbyte python=3.8 conda activate soundbyte

3. Install Dependencies

pip install -r requirements.txt

4. Install the Toolkit

pip install -e .

5. Verify Installation

soundbyte list-components
